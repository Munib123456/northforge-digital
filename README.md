# NorthForge Digital

One-page site for NorthForge Digital, a website review and optimisation service for local businesses in Newcastle and the North East.

## Run locally

```bash
npm install
npm run dev
```

Open the URL it prints (usually http://localhost:5173).

## Build for production

```bash
npm run build
```

Output goes to `dist/`.

## Before launch

In `src/NorthForgeDigital.jsx`, update:
- `EMAIL` and `LINKEDIN` (top of file)
- Replace the placeholder testimonials with real ones
